# AI-Image-Generator-
AI Image Generator built using Python. This project explores generative AI concepts like text-to-image generation, APIs, and basic UI development while building a real-world application.
  
## 🚀 Overview
This project is an AI-powered image generator that converts text prompts into images using Python and generative AI models.

## 🛠️ Tech Stack
- Python
- OpenAI / Stable Diffusion API
- Streamlit (for UI)

## 📌 Features
- Generate images from text
- Simple user interface
- Fast and easy to use

## ▶️ How to Run

1. Clone the repository
git clone https://github.com/your-username/AI-Image-Generator.git

2. Go to folder
cd AI-Image-Generator

3. Install dependencies
pip install -r requirements.txt

4. Run the app
streamlit run app.py

## 📷 Example
Enter: "A futuristic city at night"  
→ Generates an AI image
